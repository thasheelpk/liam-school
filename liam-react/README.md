## [Liam - Web App]

> 5th Generation of Blockchain Schools: An Online Learning and Creativity Platform that Allow Kids to Learn, Earn, Create, Sell and Buy their Creative Achievements

## Demo

- [Dashboard Page](http://liam-react.herokuapp.com)

## Technology
React with Axios

## Quick start

- Clone the repository

- Make sure your NodeJS and npm versions are up to date for `React 16.8.6`

- Install dependencies: `npm install` or `yarn`

- Start the server: `npm run start` or `yarn start`

- Views are on: `localhost:3000`
